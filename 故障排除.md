# 服务器启动故障排除

## 问题：无法打开 http://localhost:8080

### 步骤1：检查是否已安装Python

打开命令提示符（CMD）或PowerShell，输入：
```cmd
python --version
```

如果显示版本号（如 Python 3.x.x），说明已安装。
如果提示"'python' 不是内部或外部命令"，需要先安装Python。

### 步骤2：安装Python（如果没有）

1. 访问 https://www.python.org/downloads/
2. 下载最新版Python
3. 安装时勾选 "Add Python to PATH"

### 步骤3：进入正确的目录

在命令提示符中运行：
```cmd
cd /d D:\CodeProjects\morning-reading-zoo
dir
```

应该能看到以下文件：
- index.html
- styles\style.css
- scripts\main.js
- scripts\audio.js
- scripts\animals.js
- scripts\settings.js

### 步骤4：启动服务器

```cmd
python -m http.server 8080
```

成功后会显示：
```
Serving HTTP on 0.0.0.0 port 8080 (http://localhost:8080/) ...
```

### 步骤5：在浏览器中访问

打开浏览器（Chrome/Firefox/Edge），输入：
```
http://localhost:8080
```

## 其他解决方案

### 方案1：使用Node.js

如果安装了Node.js：
```cmd
npm install -g http-server
http-server -p 8080
```

### 方案2：使用PHP

如果安装了PHP：
```cmd
php -S localhost:8080
```

### 方案3：使用VSCode

1. 安装VSCode
2. 安装Live Server扩展
3. 打开项目文件夹
4. 右键index.html → Open with Live Server

### 方案4：直接打开文件（有功能限制）

直接双击 `D:\CodeProjects\morning-reading-zoo\index.html`

**注意**：这种方法可能无法使用麦克风功能！

## 常见错误及解决

### 错误1："端口8080已被占用"
尝试其他端口：
```cmd
python -m http.server 3000
```

### 错误2："权限被拒绝"
确保在正确的目录，并且有读取权限。

### 错误3：浏览器显示"不安全"
由于需要麦克风权限，必须使用 http://localhost 或 https，不能直接打开文件。

## 成功标志

服务器成功运行后，命令提示符会显示访问的文件列表：
```
127.0.0.1 - - [日期] "GET / HTTP/1.1" 200 -
```

这表示浏览器正在成功访问服务器。
