# 早读神器 - 声音动物园

一个通过检测环境声音来生成可爱像素风动物的Web应用，让早读变得更加有趣！

## 功能特点

- 🎤 **实时声音检测**：通过麦克风检测环境音量
- 🐾 **动态动物生成**：声音越大，动物生成越快
- 🎨 **可爱像素风**：11种Q萌的卡通风格动物
- ⚡ **实时速度调节**：4种预设速度（超快、快速、正常、缓慢）
- ⚙️ **自定义设置**：调节生成速度和动物种类
- 📱 **响应式设计**：支持投屏展示
- 💾 **本地存储**：自动保存用户设置

## 如何运行

### 方法1：使用Python（推荐）

```bash
# 确保已安装Python 3
python -m http.server 8000
```

### 方法2：使用Node.js

```bash
# 安装http-server
npm install -g http-server

# 运行服务器
http-server -p 8000
```

### 方法3：使用PHP

```bash
php -S localhost:8000
```

运行后，在浏览器中访问：`http://localhost:8000`

## 使用说明

1. **开始使用**：点击"开始使用"按钮，授权麦克风权限
2. **开始早读**：点击"开始早读"按钮，开始检测声音
3. **大声朗读**：声音越大，动物生成越快
4. **调节速度**：使用速度控制按钮（超快/快速/正常/缓慢）
5. **自定义设置**：点击"设置"按钮，调整生成参数

## 技术特点

- 使用Web Audio API进行声音检测
- Canvas绘制像素风动物
- 模块化JavaScript架构
- 响应式CSS设计
- 无需后端，纯前端实现
